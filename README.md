# Challenge Alura Store

Analisis de datos de las 4 tiendas de Alura Store para recomendar cual deberia vender el Sr. Juan.

## Contenido

- `AluraStoreLatam.ipynb`: notebook principal con el analisis completo.
- `tienda_1 .csv`, `tienda_2.csv`, `tienda_3.csv`, `tienda_4.csv`: datos de cada tienda.

## Tecnologias

- Python
- pandas
- matplotlib

## Como ejecutar

1. Abre `AluraStoreLatam.ipynb` en Google Colab o Jupyter Notebook.
2. Ejecuta las celdas en orden, de arriba hacia abajo.
3. Revisa las tablas, graficos y el informe final al cierre del notebook.

## Analisis realizado

El notebook cubre los puntos pedidos en el Trello:

- ingreso total por tienda,
- ventas por categoria,
- calificacion promedio,
- productos mas y menos vendidos,
- envio promedio por tienda,
- minimo 3 graficos para visualizar hallazgos,
- recomendacion final basada en evidencia.

## Conclusion

La recomendacion final es vender la **Tienda 4**, porque:

- registra el ingreso total mas bajo,
- tiene el ticket promedio mas bajo,
- no destaca en calificacion frente a las mejores tiendas,
- su ventaja en costo de envio no compensa su menor desempeno general.
